プライバシーポリシー

最終更新日: 2026年4月11日

1. はじめに

NextAccount（以下、「当サービス」）は、個人情報の保護を最優先事項として考えています。

2. 個人情報の定義

  • 直接識別子: 名前、メールアドレス、電話番号
  • 間接識別子: Slack User ID、ユーザー名
  • 位置情報: 経費に関連する出発地・目的地
  • 金融情報: 申請金額、支払い情報
  • 行動データ: サービス利用ログ、操作履歴

3. 個人情報の収集

  • Slack API連携：Slack User ID、ユーザープロフィール
  • ユーザー入力：経費情報、申請内容
  • Google Maps API：経路検索
  • システムログ：アクセスログ、操作履歴

4. 個人情報の利用目的

  • サービス提供：ユーザー認証、申請・承認処理、データ保存
  • サービス改善：使用状況の分析、機能改善
  • セキュリティ：不正防止、アクセス制御、監査ログ

5. 個人情報の第三者提供

提供先提供情報目的
Supabaseユーザーデータ全般データ保管
Google Maps出発地・目的地経路検索
Slack-メッセージ送信

6. セキュリティ対策

  • 通信暗号化: SSL/TLS による全通信の暗号化
  • データベース暗号化: Supabase による at-rest 暗号化
  • アクセス制御: RBAC実装
  • 認証: Slack OAuth 2.0
  • 監査ログ: すべてのアクセスを記録

7. お問い合わせ

個人情報に関するご質問: privacy@nextaccount.local

このプライバシーポリシーは日本国の個人情報保護法に準拠しています。
