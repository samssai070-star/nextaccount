1️⃣ セットアップ手順

初回登録から運用開始まで

NextAccount の初期セットアップは、4つのステップで簡単に完了できます。

ステップ1: アカウント作成

アカウント登録ページで以下の情報を入力します：

  • 企業名/部門名 - 貴社の正式名称
  • 担当者名 - セットアップを担当する方のお名前
  • メールアドレス - ログインに使用するメールアドレス
  • パスワード - 8文字以上の安全なパスワード

ステップ2: 会計年度設定

組織の会計年度を設定します。これは月次報告書や年間集計の基準となります。

💡 ヒント： 日本企業の多くは4月開始の会計年度を採用しています。カスタム会計年度にも対応しています。

ステップ3: 部門管理

組織の部門構造を登録します：

項目 説明 例
部門名 部門の正式名称 営業部、企画部、経理部
部門コード 会計管理用の部門コード D001、D002、D003
説明 部門の役割や機能 営業活動全般を担当

ステップ4: 従業員管理

チームメンバーを登録します。各従業員には以下の情報が必要です：

名前: 山田太郎 メール: yamada.taro@company.jp Slack User ID: U0A93AZ6WER (Slack連携する場合) 所属部門: 営業部
⚠️ 注意： Slack User ID は Slack のワークスペース管理画面から確認できます。ユーザーをクリックして「メンバーID」をコピーしてください。

ステップ5: Slack連携（オプション）

Slack ワークスペースと連携することで、Slack から直接領収書をアップロードできるようになります。連携手順：

  1. NextAccount の設定ページで「Slack を連携」をクリック
  2. Slack の認可画面で権限を許可
  3. 領収書受付用のチャンネルを指定

従業員一覧 メンバー名、メール、所属部門 メンバー情報の管理

クイックアクション

3️⃣ 領収書のアップロード

OCR × AI で自動処理

NextAccount は最新の OCR（光学文字認識）と AI 技術を使用して、領収書から自動的に重要な情報を抽出します。

アップロード方法

方法1: Slack から

  1. Slack で連携済みのチャンネルを開く
  2. 「+」アイコンから「ファイルを追加」を選択
  3. 領収書の写真や PDF ファイルを選択
  4. NextAccount が自動的に処理を開始

方法2: ダッシュボードから

  1. ダッシュボードの「領収書をアップロード」ボタンをクリック
  2. ファイルを選択またはドラッグ&ドロップ
  3. 処理完了を待つ

自動処理の流れ

  1. OCR 処理 - 領収書から以下を自動抽出：
    • 購入日付
    • 購入金額（税込・税抜）
    • 消費税率（8%・10%）
    • 商品/サービス内容
    • 店舗名/発行者
  2. AI 分類 - 経費科目を自動判定：
    • 交通費（ガソリン、タクシー等）
    • 食事代（接待、会議等）
    • 消耗品（文房具、PC 周辺機器等）
    • 通信費（電話、インターネット等）
  3. 仕訳生成 - 会計仕訳を自動生成
  4. 承認フロー - 管理者に通知、承認待ち
💡 ヒント： 領収書の品質が高いほど、自動抽出の精度が上がります。明るく、角度のない写真が最適です。

4️⃣ Slack 連携

チャットから直接領収書管理

Slack ワークスペースと連携することで、チームの日常的なコミュニケーション内で領収書管理ができます。

連携のメリット

  • 外出先からでもスマートフォンで領収書を送信可能
  • 承認・却下の通知がリアルタイムで届く
  • チーム全体の経費状況を可視化
  • Slack を離れずに経費処理が完結

Slack ボットコマンド

/nextaccount help - ヘルプを表示 /nextaccount status - 処理状況を確認 /nextaccount export - データをエクスポート

チャンネル設定

以下の専用チャンネルを作成することをお勧めします：

チャンネル名 用途 参加者
#経費申請 領収書アップロード用 全従業員
#経費承認 管理者用の承認通知 経理担当者
#経費レポート 月次集計レポート配信 経営者・管理層

5️⃣ 設定とカスタマイズ

組織に合わせた柔軟な設定

部門管理

部門は組織の階層構造に合わせてカスタマイズできます：

  • 部門の追加・編集・削除
  • 部門の階層化（本部 > 部 > 課）
  • 部門別予算管理
  • 部門別レポート生成

従業員管理

従業員情報を組織の変化に合わせて管理：

  • 新入社員の追加
  • 配置転換時の部門変更
  • 退職者のアカウント無効化
  • 権限設定（承認者・管理者等）

経費科目カスタマイズ

デフォルトの経費科目は以下の通りです（カスタマイズ可能）：

  • 交通費（ガソリン、電車、飛行機等）
  • 食事代（出張、接待、会議等）
  • 宿泊費
  • 消耗品
  • 通信費
  • 雑費

6️⃣ 会計ソフト連携

自動仕訳で事務作業を削減

NextAccount は主要な会計ソフトと連携し、自動的に仕訳を計上できます。

対応会計ソフト

ソフト名 対応レベル 特徴
freee 完全対応 クラウド会計ソフトの定番
MFクラウド会計 完全対応 中小企業向けの高機能ソフト
弥生クラウド 完全対応 長年の実績があるソフト

連携のステップ

  1. 会計ソフトで API キーを取得
  2. NextAccount の設定で API キーを入力
  3. 連携テストを実行
  4. 自動仕訳の設定を完成
💡 ヒント： 会計ソフトとの連携により、手作業による二重入力を完全に排除できます。

7️⃣ よくあるご質問

Q. 領収書の自動抽出の精度はどのくらい？

A. 一般的な領収書で 95% 以上の精度です。不鮮明な領収書やレシートの場合は、確認画面で手動修正が可能です。

Q. 複数の会計ソフトに同時対応できる？

A. はい、複数の会計ソフトへの同時出力に対応しています。グループ企業で異なるソフトを使用している場合も便利です。

Q. オフラインでも使用できる？

A. NextAccount はクラウドサービスのため、インターネット接続が必要です。ただし Slack では事前にファイルをダウンロードしておけば、オフライン時にもアップロード予定ができます。

Q. データのセキュリティは大丈夫？

A. 以下のセキュリティ対策を実施しています：

  • エンドツーエンド暗号化
  • 定期的なセキュリティ監査
  • ISO 27001 準拠
  • 24/7 監視

Q. 使用人数による制限はある？

A. ユーザー数による制限はありません。従業員数に応じた料金体系となっています。

Q. データはどこに保存される？

A. すべてのデータは AWS の日本リージョン（東京）のデータセンターに保存されます。

Q. 領収書の保存期間は？

A. 日本の法律に従い、7 年間保存されます。その後は自動削除されますが、事前に PDF エクスポート可能です。

Q. サポート体制は？

A. メール、チャット、電話でのサポートを提供しています。お問い合わせフォームからいつでもご連絡ください。
