1. はじめに

初期セットアップについて

NextAccount をご利用いただく際は、初期セットアップウィザードにしたがって各種設定を行っていただきます。所要時間は約10〜15分です。

セットアップの流れ

以下の情報を順番に入力していただきます：

基本情報の登録

項目企業名および部門名
項目ご担当者のお名前
項目ログイン用メールアドレス
項目パスワード（8文字以上）

会計設定

貴社の会計年度を設定していただきます。日本国内の企業の場合、一般的には4月開始となります。決算月に合わせてカスタマイズすることも可能です。

部門・従業員の登録

組織内の部門および従業員を登録します。後から追加・変更できますので、ここでは主な部門のみの登録で問題ございません。

部門コードは会計ソフト側の勘定科目コードと揃えておくと、自動仕訳の際に便利です。

2. ダッシュボード

ダッシュボードは NextAccount のメイン画面です。登録済みの部門数、従業員数、会計年度などの基本情報を確認できます。

表示される情報

項目組織名および基本情報、メンバー数
項目登録済みの部門一覧
項目従業員情報
項目ユーザーアカウント一覧

各種設定への遷移

ダッシュボードの「設定」ボタンより、以下の項目について後から変更・追加が可能です：

  • 部門の追加・編集・削除
  • 従業員情報の管理
  • 会計年度の変更

3. 領収書のアップロード

NextAccount では、領収書をアップロードすると最新の OCR 技術を用いて自動的に金額・日付・内容などを認識し、会計処理を行います。

アップロード方法

Slack 経由

Slack の連携チャンネルにファイルを追加していただくと、自動的に処理が始まります。スマートフォンから外出先での報告も簡単です。

ダッシュボード経由

ダッシュボード上の専用フォームからファイルを選択、またはドラッグ&ドロップでアップロードします。

処理の流れ

  1. OCR 認識：領収書から日付・金額・店舗名などを自動抽出
  2. AI 分類：経費科目を自動判定（交通費・食事代・消耗品など）
  3. 仕訳生成：会計仕訳を自動生成
  4. 承認：管理者の確認・承認を待機
明るく、正面からの領収書写真が認識精度を高めます。スマートフォンのカメラで問題ございません。

4. Slack 連携

Slack ワークスペースと連携することで、日常的なチームコミュニケーションの中で領収書の報告ができます。

連携のメリット

  • スマートフォンから簡単に領収書をアップロード
  • 承認結果がリアルタイムで通知される
  • Slack を離れることなく経費処理が完結
  • チーム全体の経費状況を一元管理

推奨チャンネル設定

チャンネル名#経費申請（全従業員が領収書を送信）
チャンネル名#経費承認（経理担当者が承認結果を確認）
チャンネル名#経費レポート（月次集計を経営者と共有）

ボットコマンド

/nextaccount help ヘルプを表示 /nextaccount status 処理状況を確認 /nextaccount export データをエクスポート

5. 組織設定

ダッシュボードの「設定」より、組織の構成を変更・追加していただけます。

部門の管理

以下の操作が可能です：

  • 新規部門の追加
  • 既存部門の編集・削除
  • 部門コードの設定（会計ソフト側と連動）

従業員の管理

組織の人事異動に合わせて従業員情報を更新できます：

  • 新入社員の登録
  • 配置転換時の所属部門変更
  • 退職者のアカウント無効化

経費科目

デフォルトでは以下の経費科目が用意されています：

  • 交通費・宿泊費
  • 食事代・接待交際費
  • 消耗品費
  • 通信費
  • その他雑費

6. 会計ソフト連携

NextAccount は主要な会計ソフトと連携し、抽出した情報を自動的に仕訳として計上できます。手作業での二重入力が不要になります。

対応会計ソフト

ソフトfreee（クラウド会計の定番）
ソフトMFクラウド会計（中堅企業向け）
ソフト弥生会計クラウド

連携手順

  1. ご利用の会計ソフトで API キーを取得
  2. NextAccount の設定画面で API キーを入力
  3. 連携テストを実行して動作確認
  4. 以降、自動仕訳が開始されます
部門コードと勘定科目を事前に統一しておくと、より精度の高い自動仕訳ができます。

7. よくあるご質問

OCR の認識精度について

一般的な領収書で 95% 以上の精度で認識します。不鮮明な書類の場合は、確認画面で手動修正いただけます。

複数の会計ソフトへの対応

はい、複数の会計ソフトに同時出力できます。グループ企業で異なるソフトを利用されている場合に便利です。

オフライン利用について

NextAccount はクラウドサービスのため、基本的にはインターネット接続が必要です。Slack 経由では事前にファイルをダウンロードしておくことで、後からの報告が可能な場合もあります。

セキュリティ対策

以下の対策を実施しております：

  • AWS 日本リージョンでのデータ保管
  • 定期的なセキュリティ監査
  • ISO 27001 準拠
  • 24 時間監視体制

ユーザー数に制限はあるか

ユーザー数に上限はございません。組織の従業員数に応じた料金体系になっています。

領収書の保存期間

日本の法定要件に従い、7 年間保存されます。必要に応じて PDF でのエクスポートも可能です。

サポート体制

メール、チャット、電話でのサポートを提供しています。お問い合わせフォームからいつでもご連絡ください。
