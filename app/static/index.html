// Features
選ばれる理由

Slackだけで完結する、日本で唯一の経費精算。

完全Slack完結

申請・承認・CSV出力まですべてSlack内で完結。別のWebアプリへの切り替えは一切不要。競合他社は必ずWeb画面に遷移する。

高精度OCR × Claude AI

Google Cloud VisionとClaude AIの2段階処理で、金額・日付・T番号・税区分を自動認識。誤認識があればSlackで即訂正。

9種類の会計ソフト対応

弥生・freee・MFクラウド・勘定奉行・PCA・TKC・MJS・JDL・汎用CSV。国内95%以上の会計ソフトに対応。
